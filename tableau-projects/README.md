# **Electric Vehicle Data Analysis (Tableau)**

## *Overview*
This project explores electric vehicle trends using Tableau.


## *Objective*
To analyze EV growth, adoption patterns, and regional distribution.


## *Tools Used*
- Tableau
- Data Visualization
- Dashboard Design


## *Dataset*
Contains:
- EV types (BEV, PHEV)
- Model year data
- State-wise distribution
- Vehicle manufacturers


## *Dashboard Features*
- Total EV Vehicles & Distribution
- EV Growth Over Time
- Vehicles by State (Map)
- Top Manufacturers
- CAFV Eligibility Analysis


## *Key Insights*
- EV adoption increasing yearly
- Tesla dominates market share
- Certain states have higher EV usage
- BEV vehicles dominate over PHEV
